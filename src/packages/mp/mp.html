The MP library interface for SWI-Prolog, version 1.2

This package contains the SWI-Prolog interface to the GMP library (GNU Multiple Precision arithmetic library). GMP allows for arbitrary size integers, arbitrary precision floats, and rational number arithmetic. In addition, this package includes complex arithmetic added to GMP. Whenever you run into number
representation problems in SWI-Prolog, you can use this package. MP integers are very useful as arbitrary sized bitvectors too!

The MP package can be obtained from the SWI-Prolog MP package page.
