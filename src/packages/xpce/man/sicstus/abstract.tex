This document provides an overview of XPCE/SICStus, the XPCE graphical
interface toolkit and libraries adapted for SICStus Prolog. The
XPCE/SICStus interface is based on
\url[XPCE]{http://www.swi.psy.uva.nl/projects/xpce/} version
\xpceversion{} and \url[SICStus
Prolog]{http://www.sics.se/isl/sicstus.html} version \sicsversion.
